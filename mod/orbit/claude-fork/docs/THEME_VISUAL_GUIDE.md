## Color Codes

### IBM Theme
```css
Background:     #000000 (Pure black)
Text Primary:   #00ff00 (Phosphor green)
Text Secondary: #00cc00 (Medium green)
Text Tertiary:  #008800 (Dark green)
Accent:         #00ff00 (Bright green)
```

### Win95 Theme
```css
Background:     #c0c0c0 (Windows gray)
Secondary BG:   #ffffff (White)
Text Primary:   #000000 (Black)
Text Secondary: #000080 (Navy blue)
Text Tertiary:  #808080 (Gray)
Accent:         #000080 (Navy blue)
```

## Easter Eggs

**IBM Theme:**
- Simulates authentic phosphor glow
- Matches the IBM 5151 monitor color
- Green = original IBM PC default

**Win95 Theme:**
- Button borders use exact Win95 light/shadow pattern
- #c0c0c0 is THE actual Windows 95 desktop color
- Navy blue (#000080) was Win95's primary accent

## Switch Between Themes

Click the 🎨 button in the top-right corner to access the theme menu.

All 8 themes are available:
1. **dark** - Default retro terminal
2. **light** - Clean light mode
3. **matrix** - Green Matrix rain
4. **cyberpunk** - Neon pink/cyan
5. **amber** - Amber monochrome
6. **ocean** - Blue oceanic
7. **ibm** - Classic IBM PC ← NEW!
8. **win95** - Windows 95 ← NEW!
