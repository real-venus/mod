### 2. Theme Switching

```
User Action                    System Response
в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ                    в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ

                              document.documentElement
                              .setAttribute('data-theme', 'light')
                              All CSS variables update
                              Background: dark вҶ’ light
                              Text: green вҶ’ dark green

2. Clicks вҳҖ button      вҶ’     Changes to рҹҢҷ button
                              document.documentElement
                              .removeAttribute('data-theme')
                              All CSS variables revert
                              Background: light вҶ’ dark
                              Text: dark green вҶ’ green
```
