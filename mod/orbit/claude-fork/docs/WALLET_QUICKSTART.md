# Wallet Manager - Quick Start

## 🚀 Quick Access

1. **Connect** your wallet at http://localhost:8821
2. **Click** your wallet address in the top header
3. **View** balance, transactions, and tokens

## 📊 Three Main Views

### Overview
```
💎 Balance: See your ETH balance & USD value
📍 Address: Copy your wallet address
🌐 Network: Current network & chain ID
🔑 Wallet Type: MetaMask/SubWallet/Local/Password
```

### Transactions
```
📜 History: Recent blockchain transactions
🔍 Filter: All / Send / Receive
📥 Export: Download as CSV
⚡ Details: Hash, value, status, timestamp
```

### Tokens
```
⟠ ETH: Native token balance
🪙 ERC20: Detected token balances
💰 Value: USD estimates
```

## ⚡ Quick Actions

### Copy Address
```
Click on your address → Auto copied to clipboard
```

### Export Transactions
```
Transactions tab → EXPORT CSV → Save file
```

### View Seed Phrase (Local Wallet)
```
Overview → SHOW SEED PHRASE → ⚠️ Keep secret!
```

### Refresh Data
```
Overview → REFRESH DATA → Updates everything
```

## 🔒 Wallet Types

| Type | Pros | Cons |
|------|------|------|
| **MetaMask** | Most popular, secure | Requires extension |
| **SubWallet** | Multi-chain support | Less common |
| **Local** | No extension needed | Must backup seed phrase |
| **Password** | Easy to remember | Password = full security |

## 🎯 Pro Tips

1. **Cache**: Transaction history cached for 5min (faster!)
2. **Filter**: Use filters to find specific transactions quickly
3. **Backup**: Always backup local wallet seed phrases
4. **Security**: Never share seed phrases or passwords
5. **Export**: Export transactions before clearing browser data

## ⚠️ Important

- **Seed phrases** = Full wallet access - guard with your life!
- **Passwords** (for password wallets) = Cannot be recovered
- **Local wallets** = Lost if browser data cleared without backup
- **Transaction history** = Last ~50 blocks only (use block explorer for more)

## 🔧 Troubleshooting

### Issue: Balance shows 0
**Fix**: Refresh data or check network

### Issue: No transactions
**Fix**: Wait for sync or check you're on correct network

### Issue: Tokens missing
**Fix**: Ensure you have tokens + on supported network

### Issue: Can't copy
**Fix**: Check browser clipboard permissions
