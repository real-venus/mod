## 🎯 What is LocalFS?

LocalFS is a high-performance, content-addressable local filesystem storage system that provides **IPFS-compatible CIDv0** functionality without requiring IPFS. It uses Python-to-Rust bindings for performance-critical operations like hashing and I/O.

**CIDs generated by LocalFS exactly match IPFS CIDv0** - you can compute a CID with LocalFS and it will be identical to what IPFS produces for the same content.

## ✨ Features

- **📦 Content-Addressable Storage** - Automatic deduplication using IPFS-compatible CIDv0
- **✅ IPFS CID Compatibility** - CIDs exactly match IPFS (uses UnixFS + DAG-PB encoding)
- **🦀 Rust Performance** - Optional Rust bindings for 10-100x faster hashing and I/O
- **🔒 Pinning System** - Prevent garbage collection of important content
- **🗑️ Garbage Collection** - Automatically clean up unpinned content
- **🔌 IPFS-Compatible API** - Drop-in replacement for basic IPFS operations
- **💾 Pure Python Fallback** - Works without Rust if needed
- **🚀 Zero External Dependencies** - No IPFS daemon required

## 📖 Usage

### Basic Example

```python
from localfs import LocalFS

# Initialize storage
lfs = LocalFS()  # Defaults to ~/.localfs

# Store data
data = {"hello": "world", "number": 42}
cid = lfs.put(data)
print(f"Stored with CID: {cid}")
# Output: Stored with CID: QmXx...

# Retrieve data
retrieved = lfs.get(cid)
print(retrieved)
# Output: {'hello': 'world', 'number': 42}

# Check statistics
stats = lfs.stats()
print(stats)
# Output: {'blocks': 1, 'pinned': 1, 'total_size': 35, ...}
```

### File Operations

```python
# Add a file
result = lfs.add_file('/path/to/file.txt')
cid = result['Hash']

# Retrieve file content
content = lfs.get_file(cid)
```

### Pinning

```python
# Pin content (prevent garbage collection)
lfs.pin_add(cid)

# Check if pinned
is_pinned = lfs.pinned(cid)

# Unpin content
lfs.pin_rm(cid)

# List all pins
pins = lfs.pins()
```

### Garbage Collection

```python
# Remove unpinned blocks
result = lfs.gc(aggressive=True)
print(f"Removed {result['Count']} blocks")
```

### CID Computation

```python
# Compute CID without storing
cid = lfs.cid({"some": "data"})
```

## 🛠️ API Reference

### Core Methods

| Method | Description |
|--------|-------------|
| `put(data, pin=True)` | Store data and return CID |
| `get(cid)` | Retrieve data by CID |
| `add_file(path)` | Add a file from filesystem |
| `get_file(cid)` | Get raw file content |
| `rm(cid)` | Remove content by CID |

### Pinning Methods

| Method | Description |
|--------|-------------|
| `pin_add(cid)` | Pin content |
| `pin_rm(cid)` | Unpin content |
| `pins(cid=None)` | List pinned content |
| `pinned(cid)` | Check if CID is pinned |

### Utility Methods

| Method | Description |
|--------|-------------|
| `cid(data)` | Compute CID without storing |
| `gc(aggressive=False)` | Garbage collect unpinned blocks |
| `stats()` | Get storage statistics |
| `test()` | Run self-test |
| `valid_cid(cid)` | Check if CID exists |
| `iscid(text)` | Check if text is a valid CID |

## 📁 Storage Structure

```
~/.localfs/
├── blocks/           # Content blocks (sharded by CID prefix)
│   ├── QmXx/
│   │   └── QmXx...   # Individual block files
│   └── QmYy/
├── pins/             # Pin metadata (unused, kept for future)
└── index.json        # Content index and pin registry
```

## 🔄 IPFS Compatibility

LocalFS provides **100% CIDv0-compatible** content addressing:

### Verified CID Compatibility

LocalFS CIDs **exactly match** IPFS for the same content:

```python
from localfs import LocalFS

lfs = LocalFS()

# Example: "hello world" produces the same CID as IPFS
cid = lfs.cid(b'hello world')
print(cid)
# Output: Qmf412jQZiuVUtdgnB36FXFX7xg5V6KEbSJ4dpQuhkLyfD
# (same as IPFS!)
```

**How it works:**
1. Content is wrapped in UnixFS protobuf format (Type=File)
2. UnixFS data is wrapped in DAG-PB protobuf format
3. SHA-256 hash is computed on the DAG-PB node
4. Multihash prefix (0x12, 0x20) is added
5. Base58 encoding produces the final CIDv0

### IPFS-Compatible API

```python
# Can be used as drop-in replacement
from localfs import IpfsClient  # Alias for LocalFS

ipfs = IpfsClient()
cid = ipfs.add({"data": "value"})
data = ipfs.get(cid)
```

**Compatible Methods:**
- `add()` / `put()` - ✅ CIDs match IPFS
- `get()`
- `cat()` / `get_file()`
- `pin_add()`, `pin_rm()`, `pins()`
- `add_file()` - ✅ CIDs match IPFS
- `cid()` - ✅ Compute IPFS-compatible CID without storing

**Not Supported (IPFS-specific):**
- Network operations (pubsub, swarm, etc.)
- DHT operations
- IPNS
- MFS (mutable filesystem)
- CIDv1 (only CIDv0/Qm... format)
